# Scenario 02 — contradictory (CTR catch and resolution)

**Client (fictitious):** "Talleres Roca" — family car workshop, owner Paco
(talkative, changes his mind, mixes wishes and facts). **Archetype:**
forms-or-lead-generation + booking-system hypothesis · **Profile:** standard
· **Language:** es · **Protocol:** **live-assisted** (TASK-017;
FR-019-AC-03) — the operator plays Paco; contradiction timing must react to
the agent's actual questions, so no script.

**Run:** scratch client at stage `discovery`, G0 recorded. Live sitting
~1–1.5 h + scoring. The operator improvises within the persona but MUST
plant the two contradictions at the marked moments.

## Persona baseline

Workshop with 2 mechanics; appointments by phone, always overbooked Mondays;
wants a web with an appointment-request form, services list, and "que se vea
que somos serios". Budget "unos mil quinientos". No payments online. Data:
name, phone, car plate (probe-worthy: is a plate personal data? — should
surface as data-sensitivity material or OQ, not a silent shrug).

## Planted contradictions

1. **Critical (scope/deadline)** — early (around M2): "lo urgente es el
   formulario de citas; lo del catálogo de coches de ocasión puede esperar".
   Later, when M10 starts: "el catálogo de ocasión tiene que estar desde el
   primer día, y la web antes de fin de mes". → The agent should register
   the CTR with both anchors, fire the interrupt (address within ~2 turns),
   confront neutrally, and record the resolution as a new turn.
   *Operator resolution when confronted:* citas first, catálogo "para
   cuando se pueda" — deadline stays end of month for the form only.
2. **Minor (pricing)** — in M4: "las revisiones son a precio cerrado";
   in M6: "bueno, depende del coche, se lo digo yo al cliente". → Minor CTR;
   acceptable to defer to OQ with owner, must not be silently dropped.

## Additional beats

- One emotionally loaded aside ("mi padre montó el taller hace cuarenta
  años…") — the agent should follow briefly (rapport), not steamroll.
- One "no lo sé" about hosting ("eso lo llevaba mi sobrino") → OQ with named
  owner (third-party dependency pattern).
- Operator requests early close right after resolving CTR-1 ("¿ya está,
  no?") — agent must refuse with failing criteria.

## What the run must produce

State with both CTRs (one resolved with `resolution_ref`, one open/deferred
as OQ), scope.mvp coverage `sufficient` after resolution, completion report
only when DoD truly holds.
