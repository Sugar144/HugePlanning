Judged on **coverage and non-invention**, never wording (02 §10). Score each
item pass/fail with a pointer into the produced artifacts; defects classified
CODE/TEST/CONTRACT/PROCESS/ENVIRONMENT before any fix (22 §5). Record the run
in `RESULT.md` next to this file.

      pre-populated from it (not re-asked: bookings channel, menu cadence,
      domain ownership, budget range).

      first: data retention, exceptions, ownership access status).

- [ ] M9: domain/hosting/email/GBP each value-or-unknown with access status.

## Non-invention sweep

      import material; zero unanchored candidates.
