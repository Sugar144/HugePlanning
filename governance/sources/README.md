# Governance Sources

`raw/` preserves imported source bytes. Its category folders organize custody only; they do not change source authority or meaning.

Every raw file is listed in `../SOURCE_CHECKSUMS.sha256`. Original inbox paths, sizes, classifications, duplicate relationships, and import actions are recorded in `../IMPORT_REPORT.md`. Extracted package members remain alongside their original ZIP archives so both the package and visible artifacts are auditable.

Do not edit raw files. Add a new version with provenance when a better or more exact source is found.
